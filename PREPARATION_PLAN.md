## 0. КУРАТОРСКАЯ КОНЦЕПЦИЯ: "MATERIAL MEMORY"

> *Масло → нарратив. Масло → dreamscape. Глина → жест. Древесина → новая жизнь.*

Все 4 художника трансформируют сырой материал в сосуд для памяти, истории и идентичности:
- **DUBINSKY** — масляные стики на холсте фиксируют коллективную память и поэзию быта
- **LIZI** — масло на холсте удерживает детские dreamscapes, воспоминания сквозь вуаль
- **KROTOVA** — глина хранит жест и эмоцию, керамические лица как археология чувств
- **SKNYPL** — 200-летняя грузинская сосна перерождается в функциональную скульптуру

Эта тематическая связка превращает 4-artist group show в **кураторское высказывание**. На камерной ярмарке вроде TAF (30 галерей, 20 стран) — это именно то, что запоминается.

**Контекст TAF:** Artforum описал ярмарку как "a lesson in goodwill" — здесь работают отношения и нарратив, а не агрессивный sales pressure. Наш подход: community-oriented, storytelling, гостеприимство.

**Конкуренты на TAF:** Ведущие грузинские галереи — LC Queisser, Art Beat, Window Project, E.A.Shared Space. Изучить их стиль презентации.

## 2. QR-КОДЫ + AI-ВИДЕО: КОНЦЕПЦИЯ "ART BEYOND THE FRAME"

### Идея
Под каждой работой — элегантная табличка с QR-кодом. При сканировании телефоном открывается 10-15 секундное AI-сгенерированное видео, где работа "оживает":

- **Dubinsky "New Year Party"** → персонажи начинают двигаться, ёлка мерцает
- **Dubinsky "The Shadow Plants"** → силуэт растения покачивается на ветру
- **Lizi "Firefly"** → лебеди плывут, девочка оборачивается
- **Lizi "Find Them"** → вода рябит, стулья покачиваются
- **Krotova "Ceramic Faces"** → лица медленно поворачиваются, моргают
- **SKNYPL** → модули разбираются и собираются в разные формы

### Технология генерации видео

#### Рекомендуемые инструменты:
1. **Runway Gen-3 Alpha** (runwayml.com) — лучшее качество для art-to-video
   - Загружаешь фото работы → пишешь промпт → получаешь видео
   - Стоимость: ~$76/мес (Unlimited plan)

2. **Kling AI** (klingai.com) — отличная альтернатива
   - Image-to-video с хорошим сохранением стиля
   - Бесплатный tier для начала

3. **Luma Dream Machine** (lumalabs.ai)
   - Хорошо работает с художественными стилями
   - Бесплатных кредитов хватит на тест

4. **Pika** (pika.art)
   - Быстрая генерация, хорошее качество

5. **Veo 3.1 (Google)** — если есть доступ, лучшее качество

#### Процесс:
1. Подготовить high-res фото каждой работы (без рамы, чистый crop)
2. Написать промпты для каждой работы (описание желаемого движения)
3. Сгенерировать 3-5 вариантов для каждой работы
4. Выбрать лучшие
5. Загрузить на хостинг (YouTube Unlisted или собственный сервер)
6. Сгенерировать QR-коды (qr-code-generator.com)
7. Дизайн и печать табличек

#### Дизайн табличек с QR:
```
┌──────────────────────────┐
│                          │
│  Title                   │
│  technique, size, year   │
│                          │
│      ┌─────────┐         │
│      │ QR CODE │         │
│      └─────────┘         │
│   Scan to see it move    │
│                          │
│  Art+OBJECTS              │
└──────────────────────────┘
```
- Размер: ~A6 (105x148mm)
- Материал: матовая фотобумага или пенокартон
- Стиль: тот же минимализм что в каталоге

4. **Контраст медиумов** — ваше конкурентное преимущество. Живопись + керамика + мебель в одном пространстве = запоминается. Мебель SKNYPL используйте как ФУНКЦИОНАЛЬНУЮ часть стенда (стол для каталогов, полка для керамики)

7. **Cocktail hour** — в один из вечеров устроить мини-открытие с вином прямо на стенде. Пригласить других галеристов, кураторов, пришедших на ярмарку.

9. **Коллекционерские папки** — для серьёзных покупателей: папка А4 с листом каждой работы (как в каталоге), price list, CV художников.

### D. КОНТЕНТ-МАРКЕТИНГ
10. **Behind the scenes** — снимайте ВЕСЬ процесс подготовки. Упаковка работ, перевозка, монтаж. Это контент на 2 месяца.
11. **"Pick of the Fair"** — попросите каждого художника выбрать одну работу другого участника и написать почему. Публикуйте как серию постов.
12. **QR-код на ВХОДЕ** — повесить постер Art+OBJECTS у входа на ярмарку (если разрешат) с QR ведущим на лендинг. Первое касание ещё до стенда.

## 5. 3D ВИЗУАЛИЗАЦИЯ СТЕНДА

### Что нужно от тебя:
1. **Точные размеры стенда** — длина, ширина, высота каждой стены
2. **Схема стенда** — сколько стен, есть ли перегородки, расположение входа
3. **Высота потолка** и тип освещения (трек-споты? потолочное?)
4. **Фото стенда** (если есть пустой или с прошлых лет)

### Инструменты для 3D визуализации:

#### Вариант 1: SketchUp (Бесплатный, быстрый)
- sketchup.com — web-версия бесплатна
- Идеально для быстрого макета расстановки работ
- Не нужны API ключи

#### Вариант 2: Spline (3D в браузере)
- spline.design — бесплатный tier
- Красивый рендеринг, подходит для презентации
- Не нужны API ключи

#### Вариант 3: AI-генерация 3D сцены
- **Meshy.ai** — текст/картинка в 3D модель
- **Kaedim** — 2D в 3D
- Нужны: API ключи к соответствующим сервисам

#### Вариант 4: Three.js (кастом, наш лучший вариант)
- Я могу написать интерактивную 3D визуализацию на Three.js прямо в HTML
- Пользователь крутит стенд мышкой
- Работы размещены на стенах как текстуры
- **НЕ нужны API ключи** — всё на чистом JavaScript
- Нужно только: размеры стенда от тебя

### Что мне нужно для 3D:
| Что | Зачем |
|-----|-------|
| Размеры стенда (все стены) | Построить точную модель |
| Высота потолка | Масштаб |
| План стенда (фото/схема) | Расположение стен и входа |
| Какие работы ТОЧНО едут | Финальный список для расстановки |

**API ключи НЕ нужны** — я сделаю 3D на Three.js без внешних сервисов.
